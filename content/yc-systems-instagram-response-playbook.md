# YC Systems Instagram Response Playbook

## Objective

Turn Instagram conversations into qualified leads for YC Systems.

The automation should not sound like a generic bot. It should be direct, professional, Spanish-first and focused on helping a business owner identify what they need:

- Website
- Online store
- Internal system
- CRM
- Dashboard
- SaaS MVP
- Automation
- Maintenance

## Current API Status

Publishing is active.

The current Meta token can publish content but does not yet include the permissions required for automated DMs or comment replies:

- `instagram_manage_comments`
- `instagram_manage_messages`
- `pages_messaging`
- `pages_manage_metadata`

Until those permissions are added, automated DM/comment replies should be handled through Meta Business Suite's built-in automations or manually.

## Lead Qualification Flow

### Step 1: Greeting

Hola, gracias por escribir a YC Systems.

Para orientarte mejor, dime cual de estas opciones buscas:

1. Pagina web
2. Tienda online
3. Sistema interno / CRM
4. Dashboard
5. Automatizacion
6. SaaS / app
7. Mantenimiento

### Step 2: Business Need

Perfecto. Para entender el alcance:

- Que negocio tienes?
- Que problema quieres resolver?
- Ya tienes logo, dominio o contenido?
- Lo necesitas pronto o estas explorando?

### Step 3: Route By Intent

#### Website

Una web profesional debe explicar tu oferta, generar confianza y convertir visitas en contacto.

Puedo ayudarte con landing page, web corporativa, portafolio, SEO basico, contacto y version movil.

#### Online Store

Una tienda online debe mostrar productos, facilitar pedidos y dar confianza.

Puedo ayudarte con catalogo, carrito, pedido por WhatsApp, pagos, dominio, contenido y experiencia movil.

#### Internal System / CRM

Si tu operacion depende de Excel, WhatsApp o memoria, un sistema interno puede darte mas control.

Puedo ayudarte con clientes, roles, estados, documentos, reportes, dashboard y seguimiento.

#### Dashboard

Un dashboard ayuda a decidir mas rapido porque muestra lo importante en un solo lugar.

Puedo ayudarte a visualizar ventas, reservas, tareas, estados, inventario, pagos o indicadores clave.

#### Automation

Automatizar reduce tareas repetidas, errores y tiempos muertos.

Puedo ayudarte con formularios, alertas, reportes, integraciones, flujos de trabajo y seguimiento.

#### SaaS / App

Un SaaS MVP empieza con un problema claro y una primera version medible.

Puedo ayudarte con usuarios, roles, admin, base de datos, pagos, API, app movil o panel web.

#### Maintenance

Una web o sistema necesita mejoras, contenido, soporte y ajustes constantes.

Puedo ayudarte con mantenimiento mensual, cambios, seguridad, nuevas secciones y soporte tecnico.

### Step 4: CTA

Si quieres, enviame en un solo mensaje:

- Tipo de proyecto
- Objetivo principal
- Tiempo ideal
- Presupuesto aproximado o rango

Con eso preparo una ruta inicial.

## Quick Comment Replies

### If someone comments "precio"

Gracias por escribir. Depende del alcance: web, tienda, sistema, SaaS o mantenimiento. Escribenos por DM con la palabra PROYECTO y te orientamos.

### If someone comments "info"

Claro. Dinos si buscas web, tienda online, sistema interno, automatizacion, dashboard o SaaS, y te enviamos una ruta inicial.

### If someone comments "web"

Perfecto. Una web profesional puede ayudarte a presentar tu negocio y recibir contactos. Escribenos por DM con WEB y vemos el alcance.

### If someone comments "sistema"

Si tu negocio depende de Excel o WhatsApp, un sistema interno puede darte mas control. Escribenos por DM con SISTEMA.

### If someone comments "tienda"

Una tienda online debe ayudarte a vender, no solo verse bonita. Escribenos por DM con TIENDA y vemos productos, carrito y pedidos.

### If someone comments "automatizar"

Automatizar puede ahorrar tiempo y reducir errores. Escribenos por DM con AUTO y cuentanos que proceso quieres mejorar.

## Tone Rules

- No prometer precio sin alcance.
- No sonar desesperado por vender.
- No usar telefono publico ni LinkedIn hasta que esten listos.
- Siempre llevar a una conversacion clara.
- Siempre pedir el problema de negocio.
- Siempre cerrar con una accion simple.
