# Styles

`site/styles.css` is the only stylesheet entry point used by pages.

The active system lives here:

- `tokens.css`
- `reset.css`
- `base.css`
- `layout.css`
- `components.css`
- `pages.css`
- `utilities.css`
- `responsive.css`

Legacy stylesheets are not part of the public site anymore.

When a visual issue appears, fix the owning modular file instead of stacking a stronger duplicate selector on top.
