# Styles

`styles.css` in the repository root is the only stylesheet entry point used by pages.

The active system lives here:

- `tokens.css`
- `reset.css`
- `base.css`
- `layout.css`
- `components.css`
- `pages.css`
- `utilities.css`
- `responsive.css`

`legacy-quarantine.css` is frozen compatibility. Do not add new rules there.

When a visual issue appears, remove or migrate the conflicting old rule instead of stacking a stronger rule on top.
