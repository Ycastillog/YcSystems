Términos iniciales

Términos de servicio

Estos terminos sirven como base informativa para servicios digitales de YC Systems. Los acuerdos finales de cada proyecto deben quedar definidos por propuesta, contrato o confirmación escrita.

Servicios

YC Systems puede ofrecer páginas web, tiendas online, sistemas internos, dashboards, SaaS, apps, automatizaciones, mantenimiento, soporte y consultoria digital.

Alcance y cambios

Cada proyecto debe definir alcance, entregables, tiempos, revisiónes, contenido, responsabilidades, tecnología y costo estimado. Cambios fuera del alcance pueden requerir una nueva propuesta.

Contenido y accesos

El cliente es responsable de entregar informacion, logos, textos, imagenes, productos, dominios, credenciales o aprobaciones necesarias para avanzar.

Pagos y entregas

Los pagos, depositos, fechas, entregables, propiedad de archivos y condiciones de entrega se deben acordar antes de iniciar cada fase.

Mantenimiento

El mantenimiento puede incluir cambios de contenido, mejoras visuales, correcciones, soporte básico, monitoreo y nuevas funcionalidades segun el plan acordado.
